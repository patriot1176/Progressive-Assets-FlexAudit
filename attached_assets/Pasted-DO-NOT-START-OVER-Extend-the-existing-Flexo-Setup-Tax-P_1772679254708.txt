1) Add a new optional input section directly below:

OPTIONAL — FINANCIAL & SPEED

Create a block titled:

OPTIONAL — SETUP MATERIAL WASTE

Add three optional inputs:

Avg Setup Waste (ft/changeover)

Average Press Web Width (inches)

Material Cost ($/MSI)

Notes:
• If any of these inputs are blank, do not display waste calculations.
• Web width represents the average width of presses currently running in the plant.

--------------------------------------------------

2) Use the existing model value:

Annual Changeovers Modeled

--------------------------------------------------

3) Add calculations for setup material waste.

Square Inches Waste per Setup

= Setup Waste (ft/changeover)
× 12
× Web Width (inches)

--------------------------------

MSI Waste per Setup

= Square Inches Waste ÷ 1000

--------------------------------

Waste Cost per Setup

= MSI Waste × Material Cost ($/MSI)

--------------------------------

Annual Setup Material Waste Cost

= Waste Cost per Setup × Annual Changeovers

--------------------------------------------------

Waste Cost per Setup ($)

Annual Setup Material Waste Cost ($/year)

Total Setup Cost ($/year)

Formula:

Setup Labor Cost + Annual Setup Material Waste Cost
